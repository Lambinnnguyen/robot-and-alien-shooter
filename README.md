Project Title: robot-and-alien-shooter
A 2D top-down shooter game built with Unity. The project focuses on core gameplay mechanics, enemy AI, and game state management.

✨ Core Features Implemented
- Player Controller: Developed the main character's physics-based movement and control system.

- Shooting Mechanics: Created a shooting system featuring limited ammunition and a reload function.

- Animation System: Implemented unique animations for the player character, various enemies, and special effects.

- Enemy AI: Programmed a simple AI for standard enemies that follows the player.

- Advanced Boss AI: Designed and developed a final boss with special skills, including teleportation, summoning minions, and circular bullet patterns.

- UI & Game Flow: Designed and implemented the complete UI system, including the main menu, game progression tracking, and win/loss screens.

- Audio Integration: Integrated custom sound effects and distinct background music for different game stages and events.

🛠️ Technology Stack
- Game Engine: Unity

- Language: C#

- IDE: Visual Studio

🚀 How to Run the Project
- Clone this repository to your local machine using git clone.
- Open the project folder with Unity Hub (Unity Version 202X.X or higher is recommended).
- Navigate to the Assets/Scenes folder and open the main game scene.
- Press the "Play" button in the Unity Editor to run the game.
